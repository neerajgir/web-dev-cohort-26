# Mintlify Landing Page

A responsive, high-fidelity recreation of the [Mintlify](https://mintlify.com/) landing page. This project showcases a modern, intelligent knowledge platform interface built with clean HTML, CSS, and Vite.

This project replicates the core design and structure of Mintlify's homepage, focusing on:

- **Responsive Layouts**: Seamless experience across desktop and mobile devices.
- **Modern Aesthetics**: Implementation of glassmorphism, vibrant gradients, and subtle animations.
- **Semantic HTML**: Accessible and well-structured markup.
- **Clean CSS**: Maintainable styles without heavy framework dependencies.

## ğŸ› ï¸ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 16 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/mintlify-clone.git
   cd mintlify-clone
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

### Usage

**Start the development server:**

```bash
npm run dev
```

Open your browser and navigate to the local URL provided (usually `http://localhost:5173`).

**Build for production:**

```bash
npm run build
```

**Preview production build:**

```bash
npm run preview
```
